This plugin is used to show posts under particular category in widget and in a page.

== Description ==

Category Post List Widget is a plugin which displays the most recent posts from a certain category.


= Category Post List Widget Features =


* Set the category from which the posts should come.
* Set the taxonomy for custom posts which will display on frontend.
* Set Widget dimensions to display post of selected category on front end.
* Set number of posts to show.
* Option to change ordering of posts.
* Option to change sort type of posts (can set sort by date, title, number of comments, random).
* Option to select different effects. 
* Option to show the post excerpt and how long the excerpt should be.
* Option to select different display options.
* Set Thumbnail size.
* Set different date format.
* Option to add post list in page using quick tag.
* Updated the plugin for better UX.

== Installation ==

1. Upload category-post-list-widget.zip to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Goto the Widget tab of the Presentation section and configure the widget.
4. Once plugin gets activated, quicktag will be available in posts and pages.

== Frequently asked questions ==

=1. How do I show post of particular category using this plugin?=
a) Find 'Appearance' menu in your WordPress admin panel. Click on "widget" sub-menu.
b) Drag & Drop 'Category Post List' widget in Widget area.
c) Configure the different widget options.

=2. Can I change date format?=
Yes, you can change date format. You can use any PHP standard date format e.g. F j,y for November 28, 2012.

=3. Can I change different effects?=
Yes, you can change effects of displaying your post of particular category. Effect option is provided by this category.

=4. How to add view all link?=
a) View all link option is provided in settings. You can add your custom link.

=5. How to change slider settings?=
a) Select slider in effect option.
b) You can change settings as per your requirements.

=6. Can I change excerpt length?=
Yes, you can change excerpt length. You can change excerpt length in words from Excerpt length (in words) option.
